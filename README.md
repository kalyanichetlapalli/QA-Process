# E-Commerce Dashboard

A simple static e-commerce dashboard with basic product cards, metrics, and recent orders.

## Files

- `index.html` — main dashboard page
- `styles.css` — page styling
- `script.ts` — TypeScript product card generation logic
- `package.json` — npm scripts and dependencies
- `tsconfig.json` — TypeScript compiler settings

## Run locally

1. Run `npm install`
2. Run `npm run build`
3. Open `index.html` in a browser

> The dashboard loads `script.js`, which is generated from `script.ts` during the build step.
