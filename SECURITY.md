# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it responsibly by opening a private issue or contacting the repository owner directly.

## Supported Versions

This repository uses the current supported codebase in the `master` branch.

## What to Include

Please include:
- a clear description of the issue
- steps to reproduce the vulnerability
- affected environment details
- any proof-of-concept code or logs

## Response

The maintainers will review security reports as soon as possible and acknowledge receipt of the report.
