# Contributing

## Roles and Permissions

- **Admin** — Technical Architect, Platform Leads
  - Can change settings
  - Limit this access to 2–3 people

- **Maintain Tech Leads**
  - Can manage issues, releases, and labels
  - No full admin access required

- **Write Developers**
  - Can push to feature branches
  - Can open pull requests

- **Triage QA**
  - Can manage issues, add labels, and close issues
  - No code write access
